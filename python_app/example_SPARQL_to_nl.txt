Example SPARQL query:

PREFIX concularOnt: <http://solid-pod-2/ontology#>
PREFIX con: <http://solid-pod-0/ontology#>

SELECT ?itemIdentifier ?availableFromDate
WHERE {
  SERVICE <http://localhost:3030/solid-pod-2/sparql> {
    ?item a concularOnt:InventoryItem ;
          con:itemIdentifier ?itemIdentifier ;
          con:isAvailableForReuse true .
    OPTIONAL {
      ?item con:availableFromDate ?availableFromDate .
    }
  }
}

Example NL description:
This query reaches out to the Concular Solid Pod and asks for every item that has been marked as available for reuse. For each of those reusable items, it returns the item's unique identifier and, if it exists, the date from which it became available for reuse.
